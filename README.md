# PAF-IAST AI Chatbot 🎓🤖

An intelligent chatbot for Pak-Austria Fachhochschule (PAF-IAST) 
university website that answers student queries using real data 
scraped directly from the official website.

## Features
- Web scraping of 20+ PAF-IAST pages
- Semantic search using FAISS vector database
- AI-powered answers using LLM (Groq/Gemini)
- Chat interface embedded in university website
- Covers admissions, fees, programs, scholarships & more

## Tech Stack
- Python, BeautifulSoup, Requests
- LangChain, FAISS, HuggingFace Embeddings
- FastAPI (backend), React/Streamlit (frontend)
